Fangraphs Projections Fetcher

This tool fetches 2026 projections from Fangraphs and saves them for the NPL site.

Instructions

  1. Click "Fetch Projections" below
  2. Wait for both batting and pitching data to load
  3. Click "Download JSON" to save the file
  4. Move the downloaded file to: npl-web/data/projections.json
