Scoresheet Roster Fetcher

Downloads Scoresheet roster data for the NPL site's SS Check view.

Instructions

  1. Click "Fetch Rosters" below
  2. Wait for roster and player data to load
  3. Click "Download JSON" to save the file
  4. Move the downloaded file to: npl-web/data/scoresheet_rosters.json

Note: If fetching fails due to CORS, use the Node.js script instead:
node scripts/fetch-scoresheet.js
